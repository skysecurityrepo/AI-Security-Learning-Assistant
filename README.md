# AI-Security-Learning-Assistant
A beginner-friendly AI Security Learning Assistant built using Python.

## Overview

This project demonstrates how to build a simple rule-based chatbot using Python.

The chatbot can answer questions related to:

- Artificial Intelligence (AI)
- Machine Learning (ML)
- Generative AI
- LLMs
- RAG
- MCP
- Agentic AI
- AI Security Controls
- OWASP LLM Top 10
- Threat Modeling

---

## Architecture

User Input
↓
Pattern Matching
↓
Response Selection
↓
Bot Response

---

## How to Run

Clone repository:

```bash
git clone https://github.com/YOUR_USERNAME/AI-Security-Learning-Assistant.git
```

Navigate:

```bash
cd AI-Security-Learning-Assistant
```

Run:

```bash
python chatbot.py
```
